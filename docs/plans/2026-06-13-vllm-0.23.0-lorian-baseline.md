## Warnings To Watch

No error lines were present in the current boot journal. The notable warnings
are:

- `num_speculative_tokens > 1` runs repeated forwards on the same MTP layer and
  may lower acceptance rate.
- `max_num_scheduled_tokens` is `2048` due to speculative decoding; vLLM
  suggests increasing `max_num_batched_tokens` or lowering speculative tokens /
  max sequences if performance is poor.
- `min_p` and `logit_bias` do not work with speculative decoding in this
  version.
- `NIXL` is unavailable.
- `SymmMemCommunicator` does not support device capability `12.0`.
- Custom all-reduce is disabled because P2P capability is missing or the P2P
  test failed.
- Spec decode with `FlashInferBackend` forces CUDA graph mode to `PIECEWISE`.
- The model's `generation_config.json` overrides default vLLM sampling with
  `temperature=1.0`, `top_k=20`, and `top_p=0.95`.
